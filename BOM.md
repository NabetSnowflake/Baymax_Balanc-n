# BOM
 
Lista de componentes electrónicos y materiales utilizados en la fabricación del robot auto-balancín.

## Componentes electrónicos
 
| Componente | Cantidad | Función |
|---|---|---|
| Motor DC 6V | 2 | Dar movimiento a las ruedas |
| giroscopio MPU6050 | 1 | Mediir el ángulo de inclinación del robot respecto a la vertical |
| Driver de motores L298N | 1 | Controlar el giro de las ruedas |
| Arduino Uno | 1 | Ejecutar el algoritmo PID de equilibrio autónomo |
| Portapilas para 2 baterías 12V | 1 | Sostener las baterías y conectarlas al sistema electrónico |
| Baterías 12V | 2 | Alimenar al sistema electrónico |
| Rueda 60 mm | 2 | Realizar el movimiento del robot |
| Cables | Varios | Conectar la electrónica del robot |


 
## Materiales estructurales
 
| Material | Uso |
|---|---|
| PLA | Piezas impresas en 3D |
| Acrílico 3 mm | Componentes de soporte de electrónica y partes estéticas |
